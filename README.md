## 📌 프로젝트 개요

**OWLEARN**은 아동을 대상으로 하는 **AI 기반 맞춤형 영어 동화 학습 플랫폼**입니다.  
학습자의 흥미, 수준, 난이도, 선호 그림체 등을 반영하여 **개인화된 영어 동화**를 생성하고,  
이미지·음성·퀴즈·리텔링까지 통합 제공하는 **멀티모달 교육 콘텐츠 생성 시스템**을 목표로 개발되었습니다.

본 제출용 레포지토리는 **Frontend / Backend / AI 레포지토리를 Subtree 방식으로 통합한 최종 제출 레포지토리**입니다.

---

## 📌 프로젝트 필요성

기존 교육 시스템은 모든 학습자에게 동일한 콘텐츠를 제공하기 때문에  
학습자의 수준 차이, 흥미, 몰입도 유지에 한계가 존재합니다.

OWLEARN은 아래 문제를 해결하고자 합니다:

- 학습자 수준 및 취향 반영 부족
- 이미지 오류 및 불완전한 생성물
- 생성형 AI 콘텐츠의 교육성 검증 부족
- 읽기 중심 학습의 낮은 학습 효과

---

## 📌 핵심 목표

### 🎯 1. **개인화 학습 제공**

- 학습자의 흥미·수준·선호 그림체 기반 맞춤형 영어 동화 생성
- ‘나만의 캐릭터’가 등장해 몰입감 강화

### 🎯 2. **이미지 연속성 확보**

- 장면마다 캐릭터 외형·배경 톤 유지
- 참조 이미지 기반 일관성 강화

### 🎯 3. **생성물 품질 향상**

- Role Prompting / Negative Prompt로 품질 개선
- 오류 장면 감소 및 아동 적합성 강화

### 🎯 4. **교육성 검증 및 안전한 콘텐츠 제공**

- AI 평가 4항목(교육적 가치 / 정서적 안정성 / 창의성 / 이해도) 적용
- 기준치 미달 콘텐츠 자동 폐기 후 재생성

---

## 📌 서비스 주요 흐름

### 1) **캐릭터 생성**

- 사용자가 직접 아바타를 생성
- 모든 삽화에 캐릭터를 고정 반영해 일관성 유지

### 2) **부모 및 자녀 로그인**

- 부모 계정으로 로그인
- 자녀 프로필을 선택하여 학습 진행

### 3) **맞춤 동화 생성**

- 주제·난이도·그림체 선택
- 선택 옵션 기반 베스트 맞춤형 동화 추천

### 4) **이미지 생성 및 품질 개선**

- Stable Diffusion 기반 삽화 생성
- Role Prompting / Negative Prompt 적용
- 이전 장면 참조로 연속성 확보

### 5) **동화 검증 단계**

- AI가 4가지 항목으로 재평가
- 기준 미달 시 자동 재생성

### 6) **학습 기능 제공**

- 단어 학습: 뜻·예문 자동 제공
- 리텔링: 특정 장면 설명 → 이해도 기반 크레딧 지급
- 크레딧 기반 커스터마이징 기능 제공
